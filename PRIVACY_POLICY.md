# Privacy Policy

MagicBox is currently a frontend shell maintained by LightJunction for a separate
Magisk module backend. The APK itself does not implement networking, proxy,
telemetry, or account features.

Planned frontend actions are statistics viewing, rule editing, and local issue
draft creation for MagicNet changes. Backend execution belongs to the Magisk
module.

The repository keeps attribution to the original YumeBox project where relevant,
but this app shell does not include the YumeBox mihomo runtime or proxy services.
